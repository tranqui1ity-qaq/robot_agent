🔄 执行流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第1步: PERCEPTION（感知）
      LLM 调用 perceive_environment() 获取：
      • 末端位置、夹爪宽度、物体位置、目标位置

第2步: DECISION（决策）
      LLM 推理下一步：
      • 选择工具：move_arm_to / grasp / release
      • Function Calling 编码参数

第3步: ACTION（执行）
      工具函数执行控制：
      • 末端控制 → IK 求解 → 关节命令
      • 循环 env.step() 直到到达目标

第4步: FEEDBACK（反馈）
      反馈结果给 LLM：
      • 执行状态字符串
      • LLM 评估进展

第5步: LOOP（循环）
      检查成功或继续：
      • success=true → 退出
      • 否则 → 回到第1步

⏱️ 平均耗时：35-60 秒


⚙️ 工具函数（LLM 可调用）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ perceive_environment()
  返回：场景状态（末端、夹爪、物体、目标、成功标志）

✓ move_arm_to(x, y, z)
  功能：将末端执行器移动到绝对坐标
  精度：±1 cm
  范围：X[-1.0,0.2], Y[-0.6,0.6], Z[0,1.2] m

✓ grasp()
  功能：闭合夹爪抓取物体
  宽度：3 cm（与正方体紧贴）

✓ release()
  功能：张开夹爪释放物体
  宽度：8 cm（完全打开）
